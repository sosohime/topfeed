Topfeed

React + Koa + SSR + I18n 渐进式全栈解决方案

快速入手 →

This project is heavily inspired by vue 全家桶 && egg.js

功能丰富

对Babel,Typescript,ESLint,SSR和I18n提供开箱即用的支持

易于扩展

完全兼容React和Koa的已有生态系统

约定优于配置

借鉴Egg.js的『约定优于配置』理念 ，按照一套统一的约定进行应用开发，减少配置和开发人员的学习成本。

渐进增强

推崇Vue.js提供的『渐进式增强』的主张，用户可以根据自己的需求，渐进式的对自己的页面进行优化。

# 起步

安装：

npm install -g @topfeed/topfeed-cli

创建一个项目：

topfeed init topfeed-example --type=simple
