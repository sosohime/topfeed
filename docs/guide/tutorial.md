## 骨架类型说明

你可以使用骨架类型，像下面这样

```bash
topfeed init --type=simple
```

#### 选项

| 骨架类型 |                      说明                       |
| -------- | :---------------------------------------------: |
| simple   | 含有普通页面，SSR 页面和单页 SSR 页面的实例工程 |
