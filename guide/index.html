快速开始

快速初始化

我们直接推荐使用脚手架来新建项目，只需几条简单命令，即可快速生成项目

$ npm install -g @topfeed/topfeed-cli
$ topfeed init topfeed-example --type=simple
$ cd topfeed-example
$ npm install

启动项目

npm run client:dev
npm run server:dev
open localhost:7777
